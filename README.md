# FastAPI ToDo

Приложение для управления списками задач для нескольких пользователей

## Стек технологий
- **FastAPI** (Backend)
- **SQLAlchemy 2.0** (ORM)
- **UV** (Package manager)
- **PostgreSQL** (In progress)

## Как запустить
1. Установите `uv` (если нет)
2. Выполните `uv sync`
3. Запустите: `uv run uvicorn src.main:app --reload`

## Roadmap (Бэклог)
+ Роуты только по юзер айди, для вывода данных одного пользователя
+ Роуты для работы с листами и задачи должна идти по айди пользователя
+ Возврат модели юзера из пайдентика при удачном выполнении функции.
+ Коды ответа HTTP вместо 200 по умолчанию
+ Сделать отдельный репозиторий под проект
+ Перенсти задачи по проекту в репозиторий
