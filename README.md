# FastAPI ToDo Lists

Приложение для управления списками задач для нескольких пользователей

## Стек технологий
- **FastAPI** (Backend)
- **SQLAlchemy 2.0** (ORM)
- **UV** (Package manager)
- **PostgreSQL** (In progress)

## Как запустить
1. Установите `uv` (если нет)
2. Выполните `uv sync`
3. Запустите: `uv run uvicorn src.main:app`

## Реализовано
+ Роуты только по конкретному пользователю (по user id), для вывода данных этого пользователя
+ Роуты для работы с листами и задачи по конкретному пользователю (по user id)
+ Возврат модели пользователя pydantic при удачном выполнении запроса
+ Корректные коды ответа HTTP, вместо 200 по умолчанию
+ Оформить репозиторий
+ Учет задач по проекту ведется в репозитории
